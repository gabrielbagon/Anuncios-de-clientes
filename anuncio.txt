
# Cadastro de Anúncios
 Este projeto foi desenvolvido com o propósito de testar habilidades de programação e raciocínio lógico de quem está participando do processo seletivo. 


 Features:

- Baixe o projeto Anúncios
- Ir até a pasta raiz do projeto (Anúncios)
- Abrir o arquivo anuncio.html






 Tecnologias:

 Este projeto web foi desenvolvido com as seguintes tecnologias:

- HTML
- JavaScript
- Frameworks
  - JQuery
